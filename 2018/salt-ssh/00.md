## How to manage infrastructure with salt-ssh

### Getting started.

Install salt-ssh.

```zypper in salt-ssh```


### Minimal

* create the Saltfile
```vim Saltfile ```

```
 salt-ssh:
    config_dir: etc/salt
    max_procs: 30
    wipe_ssh: True
```



# Pki configuration

```
 salt-ssh:
    config_dir: etc/salt
    max_procs: 30
    wipe_ssh: True
    pki_dir: etc/salt/pki
```
